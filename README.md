# DecodeLabs Internship Projects

This repository contains the Java projects completed during my **DecodeLabs Internship**. These projects focus on strengthening core Java concepts, Object-Oriented Programming (OOP), problem-solving, and exception handling.

## Projects

### 1. Number Game
A console-based guessing game where the computer generates a random number and the user has to guess it.

#### Features
- Random number generation
- User input validation
- High/Low hints
- Attempt counter
- Exception handling

**Concepts Used**
- Random Class
- Loops
- Conditional Statements
- Exception Handling
- Scanner Class

---

### 2. Student Grade Calculator
A Java application that calculates the total marks, average percentage, and grade based on marks entered for multiple subjects.

#### Features
- Multiple subject input
- Total marks calculation
- Percentage calculation
- Grade generation
- Input validation
- Exception handling

**Concepts Used**
- Loops
- Arrays/Variables
- Arithmetic Operations
- Conditional Statements
- Exception Handling

---

### 3. ATM Interface
A simple ATM simulation built using Object-Oriented Programming principles.

#### Features
- Deposit Money
- Withdraw Money
- Balance Inquiry
- Input Validation
- Exception Handling

**Concepts Used**
- Classes & Objects
- Encapsulation
- Constructors
- Methods
- OOP Concepts
- Exception Handling

---

### 4. Currency Converter
A console-based currency converter that converts amounts between different currencies using predefined exchange rates.

#### Features
- Multiple Currency Support
- Currency Conversion
- Input Validation
- Exception Handling
- Menu Driven Program

**Concepts Used**
- Switch Statement
- Loops
- Arithmetic Operations
- Exception Handling
- Scanner Class
